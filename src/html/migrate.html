Migrate from "Suspender" extensions ( BETA )

If you're stuck with one of the "Suspender" extensions that may soon no longer be supported by Chrome...

  1. Save a backup copy of your tabs.
    Some "Suspender" extensions have this built-in. Export a fresh copy.
  2. No, really. Back them up.
    The migration here is super simple, but just in case you should really have a backup.
  3. What's "Migrate" vs "Convert" below
    Migrate simulates Suspended tabs like you might be used to. It's old-school.
    Convert will un-Suspend your tabs, then tell Chrome to immediately Discard them. This is the preferred option, native to Chrome, but removes your "Suspended" placeholder page.

If you're using a tab Suspending browser extension, I'm not recognizing it. Please submit a new issue and include:
  1. The name of the extension you're using
  2. An example URL of a suspended tab from your browser
